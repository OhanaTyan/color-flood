# 颜色洪水战

好久之前在手机上玩到过一个名字叫“颜色洪水战”的游戏，但是现在早已找不到了。于是根据回忆中它的玩法重新还原了一下这个游戏。

## 游戏规则

游戏在一个正方形的地图上进行。地图上除了最左下角和右上角的格子以外，都涂有随机颜色。

每一回合，玩家和电脑轮流进行游戏。玩家先行，选择任意一个颜色，然后到电脑的回合，电脑选择一个颜色。然后到玩家的回合，玩家选择一个颜色，然后电脑再选择一个颜色，以此类推。

玩家在选择颜色时，不能选择与上次相同的颜色，也不能选择和上一个回合电脑选择的颜色相同的颜色。对于电脑来说也是这样。电脑每次操作时，既不能选择与上次相同的颜色，也不能选择和上一个回合玩家选择的颜色相同的颜色。

最右下角是玩家的格子，最左上角是电脑的格子。每一次玩家或电脑选择颜色后，其相应的格子，以及被该格子同化的格子，都会变成与玩家或电脑选择的颜色。与已被的同化的格子颜色相同的格子也会被同化。

最终地图上没有被同化的格子时，游戏结束。此时进入评分环节。同化格子多的一方获胜。

## 如何玩

```bash
git clone https://github.com/zhengxiaowai/color-flood.git
cd color-flood
python color-flood.py
```

目前在 Windows 系统上测试通过。未在其它平台测试。
